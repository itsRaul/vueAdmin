### 下载依赖

### 启动项目
npm start

### 打包
npm run build

```
    ├── pages           //页面
    ├── router          //路由
    ├── store           //全局状态管理vuex
    ├── style           //共用的css
    ├── utils           //常用封装工具函数    
.
```
